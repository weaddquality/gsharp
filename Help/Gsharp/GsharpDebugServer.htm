G#DebugServer

G#DebugServer is used to remote debug G# applications. It uses TCP/IP to connect to a client G# Debugger and can be used to debug other LabVIEW application instances on the same machine, LabVIEW RT, LabVIEW web services and even built LabVIEW executabes.

The G#DebugServer is located in:

<LabVIEW>/vi.lib/addons/_AddQ/G#DebugServer
