G# - Getting Started

1.       Install G# Framework and restart LabVIEW.

2.       Create a new project and save it. Right-click on MyComputer and select G# Class.

 

3.       Enter a class name and a location in the appearing dialog and press Create Class.

4.       After a few seconds the Class Icon Editor dialog appears. Select class icon header text and colors. Press Apply.

5.       Done! The class now appears in the project.

6.       Let us add a method to the class. Right-click on the class and select Add method.

7.       Enter method information in the appearing dialog and press Create method.

8.       After a few seconds the Icon Editor dialog appears. The colors are predefined by the class. Make changes if necessary and press Apply.

9.       Done! Now let us add an attribute to the class. Double-click on the ObjectAttributes.ctl in the protected folder (or right-click on the class and select Edit attributes).

10.   Add an attribute element in the cluster. Save and close the type def control.

11.   The attribute can be accessed by the utility method GetAttributes.vi. G# uses the data value reference (dvr) in LabVIEW to access the attributes.

12.   Please check the G# examples for more information about how the classes work. Good luck!
