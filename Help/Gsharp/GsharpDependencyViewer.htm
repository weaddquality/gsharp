G# Dependency Viewer

The G# Deependency Viewer is a window tool for view inheritance and dependencies.

<LabVIEW>/vi.lib/addons/_AddQ/G#DependencyViewer/G#DependencyViewer.vi

This tool is a complement to lvproj, and can be placed center/left/right side of screen as a development tool. Class inheritance are viewed in a tree with class right-click menu. The tool have refresh button for updated tree view to minimize development impact.

·         Refresh - Update tree view with current loaded classes in application instance (lvproj).

·         Root Class - Select root class for class tree (visible if G#Object been cloned).

·         Inheritance View - Class inheritance tree view. With right-click open class, class information, aggregates and attributes.

·         Always on top - Select to set this windows allways on top in labview. Will update a little rarely to not impact on development. (lvproj).

·         Left - Move window to the left side of screen.

·         Right - Move window to the right side of screen.

·         Window - Set window to center position.
