Dialog - Create Class

Creates a G# class from template.

 Class name

Name of the class to be created.

Abstract

Selects if the class should be abstract. (The constructor is declared protected scope).

Template

Class template to use.

Auto Generate Folder

Auto generated the class folder with the specified suffix.

Folder Suffix

This is the suffix to be used in the class folder.

Class folder

Folder for the class to be created.

Class description

Description of the class.

CreateClass

Creates the class.

Cancel

Cancel.

Parent Class

Parent class to inherit from. This class must be present in the active project.
