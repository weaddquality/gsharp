G# Class Item

·         Create Method – Shows a dialog to create a method.

·         Create Type Def Control – Shows a dialog to create a type def control.

·         Create Property -  Shows a dialog to create property get and set methods. If using LabVIEW 2010, this will create Property Definition folders.

·         Edit Attributes – Opens the panel of the ObjectAttributes.ctl where the attributes of the class are defined.

·         Edit Class Attributes – Opens the panel of the ClassAttributes.ctl where the attributes of the class are defined.

·         Edit Class Icon – Shows the Class Icon Editor allowing change of the class icon header and colors. These changes will be applied to all VIs in the class.

·         Change Descriptions – Shows a dialog to edit the description of the class, but also to all members of the class (VIs and controls).

·         Clone Class – Shows a dialog to clone (copy) the class. An identical copy of the class will be created with a new name, icon and location.

·         Rename Class – Shows a dialog to rename the class.

·         Delete Class – Shows a confirmation dialog to delete the class. Class will be removed from both project and disk. The class could still be in memory though until LabVIEW is restarted.

·         Change Inheritance – Shows a dialog to change the parent class to another class within the project. The class can also be converted to a base class.

·         Change Abstract – Toggles if the class should be abstract or not. The class icon will also change. If abstract, the constructor of the class will be set to protected scope and if not abstract, the constructor will be set to public scope.

·         Change Interface Inheritance – Shows a dialog for adding or removing parent interfaces to current class or interface. It will apply new relations when press OK.

·         Go to Parent Interfaces – Open parent interfaces for current class or interface.

·         Class Information – Shows a dialog containing information about the class (template, version, provider and description).
