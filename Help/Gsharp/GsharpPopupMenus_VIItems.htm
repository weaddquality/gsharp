G# VI Item

 

·         Edit Icon – Shows the VI Icon Editor dialog.

·         Edit Description – Shows a dialog to edit the VI description.

Classes only:

·         Clone – Shows a dialog to clone (copy) the selected VI as a new method within the class. Scope can be changed.

·         Rename – Shows a dialog to rename the selected VI. This option can also be used to change scope of the method and keep the same name.

·         Delete – Shows a dialog to confirm deleting of the method. The method will be removed from the class and deleted from disk as well.
