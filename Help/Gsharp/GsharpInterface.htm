IG#

IG# is the root interface of all defined interfaces. An interface is defined as a lvclass with no implementation. Any class can implement an interface by adding a wrapper class (that inherits from the interface class) and then uses aggregation to call the method in the implementing class.

The IG# class is located in:

<LabVIEW>/vi.lib/addons/_AddQ/IG#

This class is always needed then executing G# code using interfaces.
