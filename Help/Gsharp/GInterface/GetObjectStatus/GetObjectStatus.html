IG#.lvclass:GetObjectStatus.vi

Returns information about the specified object. If simple info = true, only object info is return and not the arrays of objects.

If input reference is unwired, only information about the class is returned.

 error in (no error)

The error in cluster can accept error information wired from VIs previously called. Use this information to decide if any functionality should be bypassed in the event of errors from other VIs.

The pop-up option Explain Error (or Explain Warning) gives more information about the error displayed.

    		status

    The status boolean is either TRUE (X) for an error, or FALSE (checkmark) for no error or a warning.

    The pop-up option Explain Error (or Explain Warning) gives more information about the error displayed.

    		code

    The code input identifies the error or warning.

    The pop-up option Explain Error (or Explain Warning) gives more information about the error displayed.

    		source

    The source string describes the origin of the error or warning.

    The pop-up option Explain Error (or Explain Warning) gives more information about the error displayed.

reference in

Object reference in.

simple info (F)

If true, information about all objects are returned, else only information about this object is returned.

error out

The error out cluster passes error or warning information out of a VI to be used by other VIs.

The pop-up option Explain Error (or Explain Warning) gives more information about the error displayed.

    		status

    The status boolean is either TRUE (X) for an error, or FALSE (checkmark) for no error or a warning.

    The pop-up option Explain Error (or Explain Warning) gives more information about the error displayed.

    		code

    The code input identifies the error or warning.

    The pop-up option Explain Error (or Explain Warning) gives more information about the error displayed.

    		source

    The source string describes the origin of the error or warning.

    The pop-up option Explain Error (or Explain Warning) gives more information about the error displayed.

reference out

Object reference out.

full class name

Full class name of the object including the lvlib hierarchy.

# of users

Numbers of object users.

all object names

All object names of the class.

    		Object name

last user

If true, there is only one user of the class.

Number of objects instances of the class.

class name

Name of the class, specified by the runtime object reference.

not a refnum?

If true, the object reference is not valid.

object name

Name of the object.

all refs out

All object references of the class.

    		reference out

counter

If true, user counter is enabled.
